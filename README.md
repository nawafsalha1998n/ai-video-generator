# AI Video Generator 🎬

منصة متقدمة لتوليد الفيديوهات باستخدام الذكاء الاصطناعي. حول أفكارك إلى فيديوهات احترافية من خلال النصوص والصور.

## المميزات ✨

- **توليد من النص**: اكتب وصفاً وحول أفكارك إلى فيديوهات احترافية بسهولة
- **توليد من الصورة**: أحي صورك الثابتة بإضافة حركة واقعية وسلسة
- **معاينة فورية**: شاهد الفيديو مباشرة وحمّله بسهولة عند اكتماله
- **دعم نماذج متعددة**: اختر من بين عدة نماذج ذكاء اصطناعي (Hailuo, Seedance, Kling)
- **بدون تسجيل دخول**: استخدم الخدمة مباشرة بدون الحاجة لإنشاء حساب

## التكنولوجيا المستخدمة 🛠️

### الواجهة الأمامية (Frontend)
- **React 19** - مكتبة بناء الواجهات
- **TypeScript** - لغة برمجة آمنة
- **Vite** - أداة بناء سريعة
- **TailwindCSS** - إطار عمل CSS
- **Radix UI** - مكونات واجهة مستخدم

### الخلفية (Backend)
- **Express.js** - إطار عمل Node.js
- **tRPC** - نقل البيانات الآمن
- **Drizzle ORM** - إدارة قاعدة البيانات
- **MySQL** - قاعدة البيانات

### الخدمات الخارجية
- **Replicate API** - لتوليد الفيديوهات باستخدام الذكاء الاصطناعي

## المتطلبات 📋

- Node.js 18+ 
- pnpm (أو npm/yarn)
- مفتاح API من Replicate

## التثبيت والتشغيل 🚀

### 1. استنساخ المستودع

```bash
git clone https://github.com/nawafsalha1998n/ai-video-generator.git
cd ai-video-generator
```

### 2. تثبيت المكتبات

```bash
pnpm install
```

### 3. إعداد متغيرات البيئة

```bash
cp .env.example .env.local
```

ثم قم بتحرير ملف `.env.local` وأضف مفتاح API الخاص بك:

```env
REPLICATE_API_TOKEN=your_replicate_api_token_here
```

### 4. تشغيل المشروع محلياً

```bash
pnpm dev
```

سيتم فتح الموقع على `http://localhost:5173`

## النشر على الإنترنت 🌐

### النشر على Vercel (الخيار الأفضل)

1. اذهب إلى [Vercel](https://vercel.com)
2. قم بتسجيل الدخول أو إنشاء حساب
3. انقر على "Add New Project"
4. اختر المستودع `ai-video-generator`
5. أضف متغيرات البيئة:
   - `REPLICATE_API_TOKEN`: مفتاح API الخاص بك
6. انقر على "Deploy"

### النشر على Netlify

1. اذهب إلى [Netlify](https://netlify.com)
2. قم بتسجيل الدخول أو إنشاء حساب
3. انقر على "Add new site"
4. اختر "Connect to Git"
5. اختر المستودع `ai-video-generator`
6. أضف متغيرات البيئة
7. انقر على "Deploy"

## الاستخدام 📖

### إنشاء فيديو من النص

1. انتقل إلى صفحة "ابدأ من النص"
2. أدخل وصفاً تفصيلياً للفيديو الذي تريده
3. اختر المدة والدقة والنموذج
4. انقر على "توليد الفيديو"
5. انتظر اكتمال التوليد وحمّل الفيديو

### إنشاء فيديو من الصورة

1. انتقل إلى صفحة "ابدأ من الصورة"
2. حمّل صورة
3. أدخل وصفاً لكيفية تحريك الصورة
4. اختر الإعدادات المطلوبة
5. انقر على "توليد الفيديو"
6. انتظر اكتمال التوليد وحمّل الفيديو

## الأسئلة الشائعة ❓

**س: هل يمكنني استخدام الخدمة بدون تسجيل دخول؟**
ج: نعم، الخدمة متاحة للجميع بدون الحاجة لإنشاء حساب.

**س: كم تكلفة استخدام الخدمة؟**
ج: تعتمد التكلفة على استخدام Replicate API. تحقق من أسعارهم على موقعهم.

**س: كم من الوقت يستغرق توليد الفيديو؟**
ج: عادة ما يستغرق من 1-5 دقائق حسب طول الفيديو والنموذج المستخدم.

**س: هل يمكنني تحميل الفيديو بعد إنشاؤه؟**
ج: نعم، يمكنك تحميل الفيديو مباشرة من صفحة التفاصيل عند اكتماله.
